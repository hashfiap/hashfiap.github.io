FOCO

Jul 2022 - Jan 2023

  • java
  • kotlin
  • android-studio
  • capstone

FOCO is a software for ordering food that aims to increase efficiency by allowing customers to scan QR codes on available tables, thereby reducing long wait times to order menu and secure a seat.

Developed using Java and Kotlin on Android Studio. Worked on connecting FOCO with real-time database (Firebase)
