ZeroOdds

Des 2025 - Present

  • javascript
  • web-extension
  • artificial-intelligence
  • personal

ZeroOdds currently is a prototype of a web extension that dynamically filters ads and redirects of online gambling sites, moving beyond traditional static blacklists to improve user protection. Planning on implementing Neuro-Symbolic AI system to provide explainable reasoning for blocking gambling-related web traffic.

Develop using Javascript and will utilize Neuro-Symbolic AI to inference whether the sites are considered an online gambling site by the characteristic of it.

It is currently being developed and not done yet.
